HTML & CSS com responsividade Mobile First

Nesse projeto foi desenvolvida uma página web HTML com responsividade mobile first, a partir do primeiro modelo foram desenvolvidos os layouts para Tablet e Desktop, ocultado elementos, modificandos outros e adicionando estilizações para cada tamanho de tela. Durante o projeto aprendi a:

  • Usar media queries;
  • Flex box;
  • Aplicar diferentes estilos para diferentes tamanhos de tela;
  • Retirar elementos da tela;
  • Aplicar elementos na tela de acordo com o tamanho do dispositivo;
